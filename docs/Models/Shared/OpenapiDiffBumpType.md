# OpenapiDiffBumpType

Bump type of the lock file (calculated semver delta, or a custom change (manual release))

## Example Usage

```csharp
using SpeakeasySDK.Models.Shared;

var value = OpenapiDiffBumpType.Major;
```


## Values

| Name    | Value   |
| ------- | ------- |
| `Major` | major   |
| `Minor` | minor   |
| `Patch` | patch   |
| `None`  | none    |
