# CodeSamplesJobStatus

The current status of the job. Possible values are `pending` or `running`.

## Example Usage

```csharp
using SpeakeasySDK.Models.Shared;

var value = CodeSamplesJobStatus.Pending;
```


## Values

| Name      | Value     |
| --------- | --------- |
| `Pending` | pending   |
| `Running` | running   |
